# Global-Superstore-Sales
📊 Global Superstore Power BI dashboard analyzing $12.6M sales, $1.5M profit, 178K units, and 17.4K customers (2012–2015). Covers Sales, Profit, and Quantity analysis across regions, categories, and segments. Built with Power BI and DAX using star schema, KPIs, slicers, and interactive visual breakdowns of global retail performance.
